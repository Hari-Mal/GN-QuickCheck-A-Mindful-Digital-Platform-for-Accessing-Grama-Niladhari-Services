# GN-QuickCheck-A-Mindful-Digital-Platform-for-Accessing-Grama-Niladhari-Services

This project is a user‑centered design initiative aimed at solving a critical civic problem in Sri Lanka: the stress, confusion, and inefficiency citizens face when accessing services from their local Grama Niladhari (GN) officer. Through a structured human‑centered design process, we developed GN QuickCheck – a unified digital platform (mobile app and website) that provides real‑time information, document guidance, and service clarity to citizens.

Key aspects of the project include:

Problem identification – uncovering core issues like information uncertainty, officer unavailability, process anxiety, and the lack of a centralized digital system.

User and task analysis – defining five distinct user personas (ordinary citizens, working professionals, students, elderly, GN officers) and mapping their end‑to‑end task flows.

Scenario development – illustrating a realistic user journey (e.g., a mother needing birth registration and child benefits) to guide design decisions.

Prototyping – creating high‑fidelity interactive prototypes for both mobile and web using Figma, with screens for login, registration, dashboard, service selection, document checklists, and officer availability.

Heuristic evaluation – engaging five experts (UX designer, HCI researcher, GN officer, citizen user, social service officer) to assess usability against Nielsen’s principles and domain accuracy.

Thinking aloud testing – observing five real users with varying digital literacy (young adult, parent, elderly citizen, business owner, welfare beneficiary) to gather qualitative feedback.

Results synthesis – consolidating findings into actionable recommendations (e.g., larger fonts for elderly, simpler terminology, prominent download buttons, error‑prevention cues).

The final design empowers citizens to check GN availability, view required documents, download forms, and plan a single, well‑prepared visit – reducing repeated trips, anxiety, and miscommunication. The project demonstrates proficiency in user research, persona creation, prototyping, usability testing, and iterative design, all aimed at improving digital well‑being and public service experience.
